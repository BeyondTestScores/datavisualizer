Start Building Your Survey

Create Categories

To start try creating a "category", including "subcategories". This represents the categories, measures, and scales in previous MCIEA work. You can create the entire category hierarchy this way.

You can create a category by clicking the + Create New Category link on the admin home page.

Once a category is created you can create a "subcategory" for that category by clicking the "+ Add A Subcategory" link on the page that shows that category's information.

Add Questions

Once some categories and child categories are in place you can add questions to those categories.

You can create a question either from the "+ Add A Question To This Category" link on a category page or by clicking the + Create New Question link on the admin home page.

Create Your Survey

Once you've created all of the questions in your survey, you can create the survey itself.

When a survey is created it will create a survey on Survey Monkey as well.

You can still edit categories and questions after the survey is created and it should sync with Survey Monkey.

You can create your survey by clicking the + Create New Survey link on the admin home page.
